这半年好像也没有干些什么

2023-06-08 15:55:00

6 月了，2023 也算是过半了，时间总是那么快，就在你的不经意间，悄悄流逝。

回顾下，真的感觉是浑浑噩噩中，荒废了半年。

不知是什么时候，开始有了中年危机。这并不是在开玩笑，而是实实在在地体会到了，人生的迷茫与不知所措。人们常说，三十而立，并且在互联网这样的内卷环境加持下，三十这个年龄会显得更加时间宝贵，更应该去想清楚自己想要什么，自己该做什么。

虽说疫情放开了，大家又恢复到了以前的生活状态。但不得不承认的是，经济好像并没有复苏，今年的就业应该是比去年更惨，简直是糟透了。

从今年年初开始，我断断续续有参与过一些面试。因为面试，也是提升自我的一种方式，不出去看看，永远不知道外面市场的需求怎样的人才。通过面试，我非常明确地感受并总结了下面几点：

  1. 基于疫情影响，有很多人仍处于待业状态，各个大厂也在裁人，这两波人让市场明显处于供大于求的状态，一个岗位会有多个人才进行竞争。
  2. 然而在这样的环境下，时代仍在进步，技术仍在更新，岗位所需的技术要求在逐步提高。
  3. 基于多人竞争一岗亦或是公司财政减少等原因，目前大多公司招人成本在减少。面试者很难拿到一个与期望相符的收入。

那我这半年究竟精力都放在哪里了呢，大致归纳总结了一下：

  1. 去健身房办了一张卡，强身健体，顺带减脂，已经从 66.4->61.5
  2. github 新开了一个项目 《mkTestCase》，我把我关于自动生成测试用例的一些方法，用简单地方式实现了一下。然而这只算一个 notion，连雏形都谈不上。
  3. seldom-platform 项目剩余业务功能的代码编写，并重构代码，使其具有可读性，同时完成样式调整。最终 seldom-platform 项目算是已经在试运行了，浏览地址http://seldom.testpub.cn
  4. 单词已经从 5 月开始背着走了，英语水平真是太差了，不管是为了以后的交流，还是应试，每天都要背，坚持!
  5. 历经千辛万苦，终于搞到了 openAI 的账号，现在可以愉快的使用 chatGPT 了。非常不错的生产力工具。
  6. 学习 Prompt 编写。（如何写好 Prompt 已经成为了一个分水岭。熟练掌握 Prompt 编写的人，能够很快让 ChatGPT 理解需求，并很好的执行任务。只要你的 Prompt 写的足够好，ChatGPT 可以帮你快速完成很多工作，包括写爬虫脚本、金融数据分析、文案润色与翻译等等，并且这些工作还做的比一般人出色。）

看起来好像真的没干出些啥。

最近有一次似乎可以改变人生的机会，仿佛来到了人生的抉择路口，但可惜的是却被限制卡住了，我无比失望与沮丧。

他们说能量守恒是真实存在的。人生跌跌宕宕，大起大落。我似乎是已经来到了波谷，那我的波峰在哪里呢？
